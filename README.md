# Movie rental system

API for renting and rating movies.

## Technologies used

- GO
- Fiber
- MongoDB
- Docker

## Features

- Token based(JWT) authentication system
- Admin and user routes
- Admin CRUD movies
- User can rent(24hrs), rate and search movies
